# Requirements

1. .NET >= 5.0
2. Node
3. MSSql


# initial setup
## Backend
1. Go to GroceryApp.Backend
2. run .sln file
3. open package manager console with default project 
4. Execute command:
  update-database
5. run .Net application

## Frontend
Execute following commands:
1. npm install
2. ng serve 

now client app is running
